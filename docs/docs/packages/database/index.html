Database Package

The @workloom/database package provides the PostgreSQL schema and query layer using Drizzle ORM.

Usage​

import { organizations, users, userRoles, roleMappings } from '@workloom/database'

Schema Location​

All table definitions are in packages/database/src/schema.ts. Relations are co-located with their tables.

Key Design Decisions​

  • Content-addressed versions: Package versions are immutable once published, identified by SHA-256
  • Monotonic revisions: Workspace changes increment a revision counter (no timestamp-based ordering)
  • Cascade deletes: Removing an org cascades to all its users, packages, workspaces, and role mappings
  • Role mappings: Unique constraint on (org_id, internal_role) ensures one mapping rule per role per org
