Registry Package

The @workloom/registry package provides the PackageRegistry for CRUD operations on packages and the VersionManager for immutable versioned releases with semver constraint resolution.

Usage​

import { PackageRegistry, VersionManager, MapStore } from '@workloom/registry'

// Create an in-memory store (use DrizzleRegistryStore for production)
const store = new MapStore()
const registry = new PackageRegistry(store)
const versions = new VersionManager(store)

// Create a package
const pkg = await registry.createPackage({
  name: '@workloom/standard-skills',
  orgId: 'org_123',
  description: 'Standard skill set',
  providers: ['claude', 'codex'],
  visibility: 'public',
})

// Publish a version
const ver = await versions.createVersion({
  packageName: '@workloom/standard-skills',
  version: '1.0.0',
  sha256: 'abc123...',
  channel: 'stable',
  published: true,
})

// Resolve a version constraint
const resolved = await versions.resolveConstraint('@workloom/standard-skills', '^1.0.0')
// => "1.0.0"

PackageRegistry​

CRUD service for package metadata.

MethodDescription
createPackage(input)Create a new package (errors if name exists)
getPackage(name)Fetch a package by name
listPackages(options?)List packages with optional org/visibility filters
searchPackages(query)Full-text search on name and description

VersionManager​

Manages immutable package versions with semver validation.

MethodDescription
createVersion(input)Publish a new version (errors if version exists)
getVersion(packageName, version)Fetch a specific version
listVersions(packageName, options?)List versions with channel/published filters
resolveConstraint(packageName, constraint)Find the highest published version matching a constraint

FieldTypeDefaultDescription

versionstringrequiredSemver version string
sha256stringrequiredContent hash
assetsAsset[]?[]Package assets
channelReleaseChannel?'stable'Release channel
publishedboolean?falseWhether the version is published

Semver Utilities​

The package includes a lightweight semver parser and constraint resolver.

import { parseSemVer, isValidSemVer, resolveConstraint, satisfies, parseConstraint } from '@workloom/registry'

isValidSemVer('1.2.3')       // true
isValidSemVer('1.2')         // false

const constraint = parseConstraint('^1.2.0')
const version = parseSemVer('1.5.0')
satisfies(version!, constraint!)  // true

// Resolve highest matching version from a list
resolveConstraint('^1.0.0', ['1.0.0', '1.1.0', '2.0.0'])  // => "1.1.0"

Store Interface​

Both MapStore (in-memory) and DrizzleRegistryStore (PostgreSQL) implement the RegistryStore interface:

interface RegistryStore {
  getPackage(name: string): Promise<Package | undefined>
  putPackage(pkg: Package): Promise<void>
  deletePackage(name: string): Promise<boolean>
  listPackages(options?: ListPackagesOptions): Promise<Package[]>
  getVersion(packageName: string, version: string): Promise<PackageVersion | undefined>
  putVersion(ver: PackageVersion): Promise<void>
  listVersions(packageName: string, options?: ListVersionsOptions): Promise<PackageVersion[]>
}
