Storage Package

The @workloom/storage package provides content-addressed blob storage using SHA-256 hashing, gzip compression utilities, and JSON metadata sidecars.

Usage​

import { LocalFSStore, sha256, compress, decompress } from '@workloom/storage'

// Create a filesystem-backed content store
const store = new LocalFSStore('/path/to/blobs')

// Store content — returns the SHA-256 hash
const data = Buffer.from('hello world')
const hash = await store.put(data)

// Retrieve by hash
const retrieved = await store.get(hash)

// Check existence
const exists = await store.has(hash)

// Delete
await store.delete(hash)

ContentStore Interface​

The core abstraction for content-addressed storage. All operations are keyed by SHA-256 content hash.

interface ContentStore {
  put(data: Buffer): Promise<string>
  get(hash: string): Promise<Buffer | null>
  has(hash: string): Promise<boolean>
  delete(hash: string): Promise<boolean>
}

LocalFSStore​

Filesystem-backed implementation. Blobs are stored at <root>/<hash[0:2]>/<hash[2:4]>/<hash> with a JSON metadata sidecar alongside each blob.

const store = new LocalFSStore('/var/data/workloom/blobs')

// Get the filesystem path for a hash
const path = store.blobPath(hash) // e.g. /var/data/workloom/blobs/ab/cd/abcd1234...

// Read metadata sidecar
const meta = await store.getMetadata(hash)
// { hash: "abcd...", size: 1024, createdAt: "2025-01-01T00:00:00.000Z" }

Compression​

Gzip utilities for compressing blobs before storage or transmission.

import { compress, decompress } from '@workloom/storage'

const compressed = await compress(Buffer.from('large content...'))
const original = await decompress(compressed)

Metadata​

Each stored blob has a .meta.json sidecar file containing provenance information.

import { readMetadata, writeMetadata, deleteMetadata, metadataPath } from '@workloom/storage'

// Derive sidecar path
const sidecar = metadataPath('/blobs/ab/cd/abcd1234') // => /blobs/ab/cd/abcd1234.meta.json

// Read/write metadata
const meta = await readMetadata('/blobs/ab/cd/abcd1234')
FieldTypeDescription
hashstringSHA-256 hex digest
sizenumberByte size of the blob
createdAtstringISO 8601 timestamp
